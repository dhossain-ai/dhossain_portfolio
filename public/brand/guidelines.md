# Brand Identity Guidelines

## Core Philosophy
**Minimal. Geometric. Developer-First.**
The identity reflects a synthesis of engineering precision and modern design aesthetics. It avoids corporate sterility in favor of a "indie hacker / founder" vibe.

## Logos

### Monogram (Icon)
The primary mark is the **SH Monogram**.
- **Usage**: Favicon, Avatar (GitHub/Twitter), Mobile App Icon.
- **File**: `logo-monogram.svg`
- **Construction**: Geometric "S" and "H" capable of scaling down to 16px.

### Wordmark
- **Usage**: Website Header, Resume Header.
- **File**: `logo-wordmark.svg`
- **Font Face**: Custom Geometric Sans (Inter-inspired).
- **Tracking**: Tight (-0.02em).

## Color Palette

| Usage | Color Name | Hex | RGB | Purpose |
| :--- | :--- | :--- | :--- | :--- |
| **Primary Accent** | Electric Indigo | `#6366F1` | `99, 102, 241` | Primary Actions, Brand Highlights |
| **Secondary Accent** | Vivid Violet | `#8B5CF6` | `139, 92, 246` | Gradients, Subtle Glows |
| **Background** | Obsidian | `#09090B` | `9, 9, 11` | Main Dark Background |
| **Surface** | Zinc | `#18181B` | `24, 24, 27` | Cards, Panels |
| **Text Primary** | White/Off-White | `#FAFAFA` | `250, 250, 250` | Headings |
| **Text Secondary** | Muted Gray | `#A1A1AA` | `161, 161, 170` | Body Text |

### Gradient
**"The Developer Flow"**
`linear-gradient(135deg, #6366F1 0%, #8B5CF6 100%)`

## Typography

### Primary Font
**Inter** (or **Geist Sans** / **Plus Jakarta Sans**)
- **Weights**:
  - Regular (400) - Body
  - Medium (500) - UI Elements
  - Semibold (600) - Headings
  - Bold (700) - Hero / Titles

## Usage Rules
- **Do**: Use the monogram in a square or circular container.
- **Do**: Use the gradient on dark backgrounds.
- **Don't**: Distort the logo aspect ratio.
- **Don't**: Use the gradient on low-contrast text.
